# Context Map

## Contexts

- [API](./api/CONTEXT.md) — adoption domain: animals, persons, adoptions, inspections, form templates
- [Frontend](./frontend/CONTEXT.md) — UI-specific terms for the admin SPA and form builder

## Relationships

- **API → Frontend**: The frontend consumes the API's domain concepts directly. Terms defined in the API context are canonical; the frontend context only adds UI-specific vocabulary on top.
