# Glossary

These are the key terms relevant for adoption process. They should stay consistent around the application.  

| Termin       | Description                                                          | Synonyms             | German                           |
|--------------|----------------------------------------------------------------------|----------------------|----------------------------------|
| Adoption     | The process of finding a new owner for an animal                     |                      | Vermittlung; Vermittlungsprozess |
| Organization | The club, non-profit performing the adoption or providing the animal | Club; non-profi; NGO | Organisation; Verein             |
| Person       | Some human in this application; can have different role assignments  |                      | Person                           |
| Member       | Person that is part of this applications organization                |                      | Mitglied                         |
| Mediator     | Person supervising the adoption                                      |                      | Vermittler                       |
| Adopter      | Person adopting an animal                                            |                      | Bewerber; Adoptierender          |
| Foster       | Person providing shelter for an animal until handover                |                      | Pflegestelle                     |
| Inspector    | Person performing an inspection of the adopter                       |                      | Prüfer                           |
| Tasso        | Online service for registering dogs and finding them when missing    |                      |                                  |
| User         | General term for someone using this application.                     |                      | Benutzer                         |
|              |                                                                      |                      |                                  |
